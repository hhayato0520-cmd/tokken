# 論文の骨組み: ChatGPT補完と意味ベクトル統合による多人数嗜好対応レシピ推薦手法
- アブストラクト: 
  - 本研究では、自由記述の嗜好語が持つ曖昧性・誤変換・語彙ゆれにより、多人数の嗜好統合が困難になる問題に着目する．
  - 提案手法では、まず ChatGPT によって嗜好表現の補完・正規化を行い、その後 Sentence-BERT を用いて文脈を反映した意味ベクトルへ変換する．
  - さらに、複数ユーザーの嗜好ベクトルを重み付き平均で統合し、レシピベクトルとのコサイン類似度に基づき推薦順位を算出する．
  - 本手法により、自由記述由来のばらつきを抑えつつ、複数人の嗜好を意味空間で一貫的に扱うことが可能となり、多人数に対応したレシピ推薦の精度向上に寄与する．

1. はじめに
   - P1：背景
     - 家庭など複数人でメニューを決める機会は多い
     - しかし、甘い/辛い/さっぱり等、嗜好は人によって大きく異なる
     - 多人数での意思決定は「誰かが妥協する構造」になりがちで、全員満足の実現は難しい
   - P2：自由記述の問題点
     - 利用者に嗜好を入力してもらう際、自由記述は便利だが、語彙ゆれ（白ご飯/白米/ごはん）、誤変換、曖昧語が多発する
     - 従来手法（辞書ベース、word2vec）ではこれらを統一的に扱えない
   - P3：研究の位置づけ
     - ChatGPTの補完能力により、語彙ゆれ・誤記を文脈的に整形できる
     - SBERTにより、語の意味をベクトルとして扱うことで「意味的近さ」を反映できる
   - P4：本研究の目的
     - 自由記述の嗜好を意味空間へ正規化し、複数人の嗜好を比較・統合可能な表現へ変換することを目的とする
     - 未知語を“すべて網羅”するわけではなく、曖昧さを緩和し意味的整合性を高める点に価値がある

3. 提案手法
   1. ChatGPTによる嗜好語補完・正規化
      - ChatGPTを用いて記述の揺れ・誤記・一般語の意味補完を行う
      - 例：「ブロコリ → ブロッコリー」「白ごはん → ご飯」「茶碗蒸しが好き→卵料理」
      - 完全な未知語網羅ではなく 曖昧性の緩和 が目的
   2. Sentence-BERTによる意味ベクトル化
      - ChatGPT補完後の嗜好語をSBERTで埋め込み
      - 文脈を考慮したベクトルにより、意味距離が適切に反映される
      - 「こってり」「濃厚」「油多め」は距離が近く、「さっぱり」とは距離が遠い    
   3. 多人数嗜好の統合（重み付き平均）
      - 個人ベクトル v_i を集約し、V = Σ w_i v_i で統合ベクトルを作る
      - 好き嫌いが激しいユーザーへ重み付けするなど柔軟な設計が可能
   4. レシピベクトルとのコサイン類似度計算
      - レシピ情報も同様にSBERTで埋め込み
      - 統合嗜好ベクトル V と比較し類似度スコアを算出
      - スコアの高い上位Kレシピを推薦
   5. 根拠語の抽出（説明性）
      - どの嗜好語がレシピとマッチしたか根拠語を併記
      - 透明性とユーザー納得感を強化
